Flange

Declarative configuration for networks

Flange presents a different approach to network configuration and function placementthat abstracts away implementation specifics; freeing administrators to focus on the semantics of the network.

Flange uses flows, negated flows and functions in composition through set operations and logical transformations to generate network migrations for network configurations.These configurations are rendered through backend compilers for target environments such as Openflowand the eXtensibleSession Protocol (XSP).
